## Proyectos destacados

<details open>
<summary><b>🎯 MejorDicho!</b> — Plataforma de cultura popular colombiana</summary>

<br/>

Plataforma para **preservar y compartir la cultura popular colombiana** (dichos, refranes y piropos), organizada por los 33 departamentos del país. Diseñada, desarrollada y desplegada de forma **individual, de punta a punta**.

| Aspecto | Detalle |
|---|---|
| **Stack** | Next.js 14 (App Router) · FastAPI · PostgreSQL |
| **Escala** | Cobertura de los 33 departamentos de Colombia |
| **Performance** | PWA instalable · Nginx + Gunicorn · CI/CD |
| **Seguridad** | Backend FastAPI con validación server-side |
| **Impacto** | Producto público y comunitario con IA |
| **Repositorio** | [mejor-dicho.com](https://mejor-dicho.com) |

Incluye un **mapa interactivo** de Colombia para explorar por región, **búsqueda semántica con IA (Claude)**, y un sistema social de **reacciones, comentarios y ranking**. Un proyecto end-to-end que abarca desde el diseño de la base de datos hasta el despliegue en producción.

</details>
